üß† 0th Class: What is Algebra?

Exploring the idea of algebra, equivalence, sets, and foundational arithmetic in \(\mathbb{N}\).

Let \( X (\neq \phi) \) be a set. A relation '~' on \( X \) is called an equivalence relation if:

  • \( x \sim x \quad \forall x \in X \)
  • \( x \sim y \Rightarrow y \sim x \)
  • \( x \sim y \) and \( y \sim z \Rightarrow x \sim z \)

Equivalence class: For \( x \in X \), the equivalence class of \( x \) is: \[ [x] = \{ y \in X \mid x \sim y \} \]

Note: For \( x, y \in X \), either \( [x] = [y] \) or \( [x] \cap [y] = \phi \).
Hence, \( X \) is the disjoint union of equivalence classes.

Check: If \( a \in [x] \), then \( [x] = [a] \)

Successor Operation on Sets

Define \( S(A) = A \cup \{A\} \)

  • \( 1 \rightarrow S(\phi) = \{\phi\} \)
  • \( 2 \rightarrow S(\{\phi\}) = \{\phi, \{\phi\}\} \)
  • \( 3 \rightarrow S(\{\phi, \{\phi\}\}) = \{\phi, \{\phi\}, \{\phi, \{\phi\}\} \)

If \( 0 \rightarrow \phi \), then:

  • \( 1 \rightarrow \{0\} \)
  • \( 2 \rightarrow \{0, 1\} \)
  • \( 3 \rightarrow \{0, 1, 2\} \)
  • \( \mathbb{N} = \{0, 1, 2, 3, \ldots\} \)

Remark

Every natural number is the set of all its predecessors.

Define Addition on \( \mathbb{N} \)

  • \( n + 0 = n \quad \forall n \in \mathbb{N} \)
  • \( n + S(m) = S(n + m) \) ‚Äî recursive definition

Example: \( n + 1 = n + S(0) = S(n + 0) = S(n) \)

Peano's Axioms

  • \( 0 \) is a natural number.
  • The successor of a natural number is a natural number.
  • \( 0 \) is not the successor of any natural number.
  • \( S(n) = S(m) \Rightarrow n = m \)
  • Induction: If true for 0 and true for \( n \Rightarrow \) true for \( S(n) \), then it holds for all \( n \in \mathbb{N} \).

Properties of \( \mathbb{N} \) with respect to '+' operation

  • \( n + (m + p) = (n + m) + p \)
  • \( n + 0 = 0 + n \)
    (Forms a monoid with 1, 2)
  • \( n + m = m + n \)
    (Forms an abelian monoid)
  • \( n + m = n + p \Rightarrow m = p \)
    (Cancellative abelian monoid)

Embedding via Injection

Let \( X, Y \) be sets. Suppose \( \phi: X \rightarrow Y \) is injective.
We treat \( X \) as a subset of \( Y \) by identifying \( x \sim \phi(x) \), where \( \phi(x) \in Y \).

Equivalence Classes and Integers

Let \( X = \mathbb{N} \times \mathbb{N} \). Define a relation \(\sim\) on \(X\) by

\((a,b) \sim (a',b')\) iff \(a + b' = b + a'\).

Check: This is an equivalence relation.

\(Y =\) set of equivalence classes \(= \{[(a,b)] \mid a, b \in \mathbb{N}\}\)

Define addition on \(Y\) by

\([(a,b)] + [(c,d)] = [(a+c), (b+d)]\)

This is well defined if

\([(a,b)] = [(a',b')]\) and \([(c,d)] = [(c',d')]\)

\(\Rightarrow [(a,b)] + [(c,d)] = [(a',b')] + [(c',d')]\)

Note: This can be easily checked.

\(Y =\) set of equivalence classes. From now on, write \(Y = \mathbb{Z}\).

\((\mathbb{Z}, +)\) satisfies:

  • (i) Associativity
  • (ii) \([(a,b)] + [(0,0)] = [(a,b)]\)
  • (iii) Abelian property

Remark: For any \(a \in \mathbb{N}\), \([(a,a)] = [(0,0)]\)

Define \(\phi: \mathbb{N} \rightarrow \mathbb{Z}\) such that \(\phi(n) = [(n,0)]\).

  • \(\phi\) is injective
  • \(\phi(a) + \phi(b) = \phi(a+b)\)

We shall identify \(a \in \mathbb{N}\) with its image \([(a,0)]\). We shall write \(-a\) for \([(0,a)]\).

Integer Properties

\([(a,b)] = [(a,0)] + [(0,b)] \Rightarrow [(a,b)] = a - b\)

Conclusion: Every integer is the difference of two natural numbers.

Note: Important property of \(\mathbb{Z}\):

\([(a,b)] + [(b,a)] = [(0,0)]\)

Additive Inverse: Every element of \(\mathbb{Z}\) has another element in \(\mathbb{Z}\) such that their sum is \([(0,0)]\).

Multiplication

Multiplication on \(\mathbb{N}\):

  • \(n \times 0 = 0 \quad \forall n \in \mathbb{N}\)
  • \(n \times S(m) = n \times m + n\)

The multiplication extends to \(\mathbb{Z}\) and can be checked easily.

Construction of Rational Numbers

Let \(X = \mathbb{Z} \times (\mathbb{Z} \setminus \{0\})\).

Define a relation on \(X\) by \((a,b) \sim (a',b')\) iff \(ab' = ba'\).

This is also an equivalence relation on \(X\).

\(Q =\) set of equivalence classes \(= \{[(a,b)] \mid a \in \mathbb{Z}, b \in \mathbb{Z} \setminus \{0\}\}\)

Define \(+\) on \(Q\) by:

\([(a,b)] + [(c,d)] = [(ad + bc, bd)]\)

Exercise: Check that if \([(a,b)] = [(a',b')]\) and \([(c,d)] = [(c',d')]\), then

\([(ad + bc, bd)] = [(a'd' + b'c', b'd')]\)
